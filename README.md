# AI Automation Portfolio

This repository serves as a central showcase of AI-powered automation projects built using **n8n**, **OpenAI**, **Google Workspace**, and other no-code/low-code tools. Each project below links to its own dedicated repository with full documentation, setup instructions, and workflow exports.

---

## Featured Projects

### 1. PureSmile — Dental Hospital Patient Automation System
End-to-end patient communication system for a dental clinic: lead capture, appointment reminders, booking confirmations, AI-personalised patient reactivation emails, and an error-handling workflow.

**Tech Stack:** n8n, Google Sheets, Google Forms, OpenAI GPT-4o-mini, Gmail API, HTML/CSS/JS

🔗 https://github.com/adeniranprecious002-ux/pure_smile

---

### 2. HireIQ Automation — AI Resume Screening Engine
Backend automation that receives resumes and a job description, extracts text from PDFs, scores and ranks candidates using GPT-4o, and logs structured results to Google Sheets via a REST webhook.

**Tech Stack:** n8n, OpenAI GPT-4o, Google Sheets, Webhook API, PDF extraction

🔗 https://github.com/adeniranprecious002-ux/hireiq-automation

---

### 3. GrowthPath — AI Lead Generation & Qualification System
A fully automated, multi-channel lead generation system that engages leads via an AI chat agent, scores them as Hot/Warm/Cold, books meetings for hot leads, logs data to a CRM, and runs automated email follow-up sequences with team notifications.

**Tech Stack:** n8n, AI Chat Agent (Voiceflow/Chatbase), Airtable/Google Sheets, Google Calendar, Zapier, Slack

🔗 https://github.com/adeniranprecious002-ux/growthpath-ai-lead-system

---

### 4. Email Management Automation Workflow
AI-powered email triage system that monitors a Gmail inbox, classifies incoming emails by department using GPT-4o-mini, routes notifications to the correct Slack channels, and automatically acknowledges customer service emails.

**Tech Stack:** n8n, Gmail API, Slack API, OpenAI GPT-4o-mini, LangChain Text Classifier

🔗 https://github.com/adeniranprecious002-ux/email_management_workflow
